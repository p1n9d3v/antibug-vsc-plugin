# Antibug


커스터마이징한 Slither를 사용하여 Contract 정적 분석 및 보고서 생성, 자체 로컬 체인을 이용하여 시뮬레이션 및 테스트가 가능한 Visual Studio Extension

## 기술
- Visual Studio Extension
- Ethereum JS
- Jquery
- EJS
- ethers.js
- Slither
